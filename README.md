# Unity, Arduino, Oculus를 이용한 인터랙티브 미디어 아트웍



## Arduio

- 인풋을 받는 컨트롤러
- 벨로스탯을 이용하여 발의 무게중심과 압력값을 입력받을 수 있는 발판 컨트롤러



## Unity

- 좌 우 무게중심, 점프 인터랙션을 활용한 게임 제작



## Oculus Quest1

- 게임 배포 매체



## 맵 2탄

- 심해 맵처럼 처음과 끝이 있는 맵
- 코너링이 들어가도 좋고
- 저절로 내려갈 수 있게 경사가 좀 있게 제작

- 파닥파닥
- 슬라이딩
